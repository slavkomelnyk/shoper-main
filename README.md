# shoper-main

## install
1. install `requirements.txt`
2. migrate `python3 manage.py migrate`
3. createsuperuser `python3 manage.py createsuperuser`
4. runserver `python3 manage.py runserver` or `python3 manage.py runserver <ip address>:8000`
